Each SDK is installed primarily **through a package manager**. Some distributables, such as firmware, are downloaded directly from GitHub Releases.

## Package managers

| SDK | Install | Host |
|---|---|---|
| **Python SDK** | `pip install hapbeat-python-sdk` (import name is `hapbeat`) | [PyPI](https://pypi.org/project/hapbeat-python-sdk/) |
| **JavaScript SDK** | `npm install @hapbeat/sdk` | [npm](https://www.npmjs.com/package/@hapbeat/sdk) |
| **Unity SDK** | UPM → "Add package from git URL" → `https://github.com/Hapbeat/hapbeat-unity-sdk.git` (pinning a tag is recommended) | [GitHub Releases](https://github.com/Hapbeat/hapbeat-unity-sdk/releases/latest) |

Setup guides:

- [Python SDK Getting Started](/en/docs/sdk-integration/python-sdk/getting-started/)
- [JavaScript SDK Getting Started](/en/docs/sdk-integration/js-sdk/getting-started/)
- [Unity SDK Installation](/en/docs/sdk-integration/unity-sdk/installation/)
