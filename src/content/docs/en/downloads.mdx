Every distributable is provided through GitHub Releases. The links below always point to the **latest release** (GitHub automatically redirects `/releases/latest` to the newest version).

## Unity SDK

Installable via the Unity Package Manager (UPM).

<DownloadItem
  name="Hapbeat Unity SDK (UPM tarball)"

/>

UPM registration URL: `https://github.com/Hapbeat/hapbeat-unity-sdk.git` (pinning a tag is recommended)

Setup guide: [Unity SDK Installation](/en/docs/sdk-integration/unity-sdk/installation/)

## Device Firmware

Firmware for the Hapbeat device itself (ESP32-S3). It is normally bundled with the tooling, so downloading it separately is for advanced users.
