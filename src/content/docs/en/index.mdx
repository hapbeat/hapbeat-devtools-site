## Where to start

<CardGrid>
  <Card title="Just got a Hapbeat" icon="rocket">
    Follow [Getting Started](/en/docs/getting-started/): install Manager, flash firmware, build your first Kit in Studio.
  </Card>
  <Card title="Want to create content" icon="pencil">
    Open [Hapbeat Studio](/studio/) and follow the [Studio guide](/en/docs/studio/getting-started/) to edit waveforms, LEDs, display, and export a Kit.
  </Card>
  <Card title="Want to integrate with a game" icon="puzzle">
    Install the [Unity SDK](/en/docs/unity-sdk/installation/) and reference the [Contracts spec](/en/docs/contracts/overview/). Unreal is [in development](/en/docs/unreal-sdk/coming-soon/).
  </Card>
  <Card title="Want to dive into the hardware" icon="setting">
    Check [Architecture & concepts](/en/docs/concepts/) and [Contracts overview](/en/docs/contracts/overview/) for protocol, addressing, message format.
  </Card>
</CardGrid>
