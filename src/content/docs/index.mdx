## まず何をすればいい？

<CardGrid>
  <Card title="Hapbeat を手に入れたばかり" icon="rocket">
    [Getting Started](/docs/getting-started/) に従い、Manager を入れてファームを書き込み、Studio で最初の Kit を作ってみる。
  </Card>
  <Card title="コンテンツを作りたい" icon="pencil">
    [Hapbeat Studio](/studio/) をブラウザで開き、[Studio ガイド](/docs/studio/getting-started/) に沿って波形・LED・ディスプレイを編集して Kit を書き出す。
  </Card>
  <Card title="ゲームに組み込みたい" icon="puzzle">
    [Unity SDK](/docs/unity-sdk/installation/) を導入し、[Contracts の仕様](/docs/contracts/overview/) を参照しながら実装する。Unreal は[準備中](/docs/unreal-sdk/coming-soon/)。
  </Card>
  <Card title="ハードウェアを深く知りたい" icon="setting">
    [アーキテクチャ](/docs/concepts/) と [Contracts 概要](/docs/contracts/overview/) で通信・アドレシング・メッセージ仕様を確認する。
  </Card>
</CardGrid>

## ダウンロード
