各 SDK は基本的に **パッケージマネージャー経由でインストール** します。ファームウェアなど一部の配布物は GitHub Releases から直接入手します。

## パッケージマネージャー

| SDK | インストール | ホスト |
|---|---|---|
| **Python SDK** | `pip install hapbeat-python-sdk`（import 名は `hapbeat`） | [PyPI](https://pypi.org/project/hapbeat-python-sdk/) |
| **JavaScript SDK** | `npm install @hapbeat/sdk` | [npm](https://www.npmjs.com/package/@hapbeat/sdk) |
| **Unity SDK** | UPM →「Add package from git URL」→ `https://github.com/Hapbeat/hapbeat-unity-sdk.git`（タグ指定推奨） | [GitHub Releases](https://github.com/Hapbeat/hapbeat-unity-sdk/releases/latest) |

各 SDK の導入手順:

- [Python SDK Getting Started](/docs/sdk-integration/python-sdk/getting-started/)
- [JavaScript SDK Getting Started](/docs/sdk-integration/js-sdk/getting-started/)
- [Unity SDK インストール](/docs/sdk-integration/unity-sdk/installation/)
