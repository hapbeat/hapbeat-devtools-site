# docs/ — ユーザー / 開発者向けドキュメント

このディレクトリは、本リポジトリの **公開ドキュメント置き場** である。

- 想定読者: SDK 開発者、Hapbeat を利用するユーザー、外部統合者
- 集約先: [hapbeat-devtools-site](https://devtools.hapbeat.com/) が build 時に自動取得し、`/docs/contracts/` 等の URL で公開する

## このリポジトリでの分類

| ディレクトリ | 用途 | 公開対象 |
|------|-----|--------|
| `docs/` | ユーザー / SDK 開発者向け解説（このディレクトリ） | ◯ portal site に掲載 |
| `specs/` | 規範的な仕様文書（プロトコル・スキーマ定義） | ◯ portal site に掲載 |
| `schemas/` | JSON Schema 等の機械可読仕様 | ◯ |
| `dev-notes/` | （存在する場合）内部実装の知見・履歴 | ✗ portal には載らない |

## 書くものの例

- "Event ID とは何か" の概念解説
- プロトコル全体像のチュートリアル
- 用語集
- 規範的でない補足解説

規範的な仕様（"このフィールドは必須" 等）は `specs/` に書くこと。
