Manager は左サイドバーに各機能タブが並ぶ構成です。デバイスは自動検出され、カードベース UI で一覧されます。

## サイドバーの構成

| タブ | 用途 |
|------|-----|
| **Devices** | 検出されたデバイスをカード形式で一覧表示。接続状態、バッテリー、ボリュームなど |
| **Kit** | デバイスにインストール済みの Kit 一覧、転送、削除 |
| **Firmware** | ファームウェア書き込み、シリアル接続でのデバイス初期設定 |
| **Live Audio** | PC 音声を UDP でストリーミング（リアルタイム触覚化） |
| **Test** | 簡易再生テスト（Kit 内クリップを直接 trigger） |
| **Settings** | アプリ設定、ログレベル、ネットワークオプション |

## デバイス検出のしくみ

Manager 起動時に自動で以下の検出を並行実行します。

- **mDNS (Bonjour)**: `_hapbeat._udp.local` の応答を監視
- **UDP broadcast PING**: `255.255.255.255:9000` に PING を送り、応答してきたデバイスをリストに追加
- **Serial スキャン**: USB CDC ポートを定期的に列挙し、Hapbeat デバイスを識別

検出されたデバイスは Devices タブにカードとして表示され、オフラインになると自動的に消えます（即時削除）。

## デバイスカードの情報

各カードには以下の情報が表示されます。

- デバイス名（例: `hapbeat-duowl-01`）と group ID
- 接続状態（オンライン / オフライン）
- 現在の Volume レベル（0-128 / %）と device_wiper（強度補正値）
- バッテリー残量（SOC %、ESP32 + BQ27220 ガス計から取得）
- IP アドレス / Wi-Fi RSSI

カードをクリックすると右側に詳細パネルが開き、Kit 一覧・LED 状態・OLED 表示プレビュー・各種設定が確認できます。

## ログモニタ

下部のログドロワーには接続イベント・送信コマンド・エラーが時系列で表示されます。デバッグ時は Settings タブからログレベルを `DEBUG` に上げてください。
