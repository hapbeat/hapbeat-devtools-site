# docs/ — ユーザー向けドキュメント

このディレクトリは、本リポジトリの **ユーザー向け公開ドキュメント** 置き場である。

- 想定読者: Hapbeat デバイスを使う一般ユーザー、上級者（自前ビルドする人）
- 集約先: [hapbeat-devtools-site](https://devtools.hapbeat.com/) が build 時に自動取得し、`/docs/firmware/` の URL で公開する

## ディレクトリ分類

| ディレクトリ | 用途 | 公開対象 |
|------|-----|--------|
| `docs/` | ユーザー向け解説（このディレクトリ） | ◯ portal site に掲載 |
| `dev-notes/` | （存在する場合）内部実装の知見・履歴 | ✗ portal には載らない |
| `hardware/` | 回路設計・基板データ・ハードウェア仕様 | 公開設計（中身は技術ユーザー向け） |

## 書くものの例

- Wi-Fi 設定手順（serial / SSID 切替）
- OTA 更新の流れ
- LED / ボリューム / ボタンのカスタム設定方法
- ディスプレイ表示のカスタマイズ
- バッテリー駆動時の注意
- トラブルシューティング（接続できない / 起動しない 等）

PlatformIO で自前ビルドする上級者向け手順は `dev-notes/` に書くこと。
