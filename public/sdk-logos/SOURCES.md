# SDK ロゴの出所と商標表示

トップページ「対応 SDK」セクション（`src/components/SupportedSdks.astro`）で使用する
各 SDK/プラットフォームのロゴ。Hapbeat が対応する環境を示す目的（指名的表示）で、
各社の公式または標準的なロゴ SVG を配置している。各ロゴは各権利者の商標であり、
所有権は各権利者に帰属する。

| ファイル | 対象 | 出所 | 商標権者 |
|---|---|---|---|
| `python.svg` | Python | Wikimedia Commons `Python-logo-notext.svg`（公式カラー版） | Python Software Foundation |
| `javascript.svg` | JavaScript | Wikimedia Commons（標準の "JS" バッジ, Chris Williams 作） | — |
| `unity.svg` | Unity | Simple Icons (simpleicons.org) の Unity マーク | Unity Technologies |

差し替え・追加時は、可能な限り公式ブランドリソースまたは権威あるソース
（Wikimedia Commons / Simple Icons 等）の実ファイルを使うこと。手描き再現はしない。
