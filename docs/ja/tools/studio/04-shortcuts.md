`?` の **操作説明** ボタン (Kit タブ上部) をクリックすると、Studio 内に最新のチートシートがポップオーバー表示されます。本ページはそのリファレンス版です。

## Library パネル（選択中）

| キー | 動作 |
|------|-----|
| `↑` / `↓` | 前後のクリップに移動 |
| `←` / `→` | Library 側 Amp (libraryIntensity) を ±5% 調整 |
| `Space` | 選択中クリップを再生 / 停止 |
| `Enter` | 選択中クリップをアクティブ Kit に追加 |

## Kit Editor パネル（選択中）

| キー | 動作 |
|------|-----|
| `↑` / `↓` | 前後の Kit Event に移動 |
| `←` / `→` | Event intensity を ±5% 調整 |
| `Space` | 選択中 Event を再生 / 停止 |
| `Delete` / `Backspace` | 選択中 Event を Kit から削除 |

> 💡 `←` / `→` で Amp スライダーが native 値変更にも反応してしまわないよう、スライダー上にフォーカスが残っている場合は ↓ / ↑ で blur してからカード移動します。

## 編集モーダル

| キー | 動作 |
|------|-----|
| `Tab` | 次のフィールドにフォーカス |
| `Enter` | 確定して閉じる |
| `Esc` | キャンセルして閉じる |

## マウス操作 (Library)

| 操作 | 動作 |
|---|---|
| カードクリック | カードを選択 |
| `▶` ボタン | 再生 / 停止 |
| `+ Kit` ボタン | アクティブ Kit に追加 |
| ドラッグ | Kit パネルにドロップして追加 |
| `Edit` ボタン | 名前 / Note / タグの編集モーダル |
| `Swap` ボタン | Name ↔ Note を入れ替え |
| `×` ボタン | clips/archive/ に退避 (ファイル自体は残る) |

## マウス操作 (Kit)

| 操作 | 動作 |
|---|---|
| カードクリック | カードを選択 |
| `> FIRE` / `♪ CLIP` / `>♪ BOTH` | モード切替 (ヘッダーの「モード説明」参照) |
| `Edit` ボタン | Event 詳細モーダル |
| `Swap` ボタン | clipName ↔ Note を入れ替え (eventId 再合成 + ファイル rename も実行) |
| `×` ボタン | Kit から削除 |

Kit 内のカード並び順は **ヘッダーの Sort セレクト** で決まります (名前 / 更新日時 / 長さ / 追加順)。設定は localStorage で永続化されます。

## 一括操作

| 場所 | 操作 |
|---|---|
| Library | **Amp Preset バー** で Library 全体の Amp プリセット保存 / 読込 |
| Kit ヘッダー | **一括変更…** セレクトで Kit 内全 Event の Mode を FIRE / CLIP / BOTH に一括切替 |

## 設定

ショートカットはカスタマイズ不可です。
- パネル内にカーソルがある / フォーカスしているときだけ反応します
- テキスト入力中は無効
- Amp スライダー上では `←` / `→` はスライダー値の微調整が優先されます (`↑` / `↓` でカード移動)
